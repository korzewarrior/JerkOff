A Chrome extension that intercepts your angry comments and rewrites them into something supportive using AI. Because the internet has enough jerks already.

// How It Works

Three steps to not being a jerk

01

Type your angry comment

Go ahead, write that furious reply. Type out every bitter, sarcastic word you're thinking. Get it all out.

02

Click "Jerk Off"

Hit the button that appears next to any text input. The extension sends your draft to an AI for a full de-jerking.

03

Post the nice version

Review the rewritten comment. It keeps your intent but strips the venom. Click accept, and be the bigger person.

Download & Install

JerkOff isn't on the Chrome Web Store (yet). Grab the latest release and sideload it in a few easy steps.

Download Latest Release

Manual Install (Sideloading)

  1. Download and unzip the release to a folder on your computer.
  2. Open Chrome and go to chrome://extensions
  3. Enable Developer mode (toggle in the top right corner).
  4. Click "Load unpacked" and select the unzipped extension folder.
  5. Click the extension icon, enter your API key, and you're ready to de-jerk.
