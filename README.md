# JerkOff

Chrome extension that rewrites your aggressive comments into supportive ones using AI.

## Install

1. [Download the latest release](https://github.com/korzewarrior/JerkOff/releases/latest)
2. Unzip, go to `chrome://extensions`, enable Developer mode
3. Click "Load unpacked", select the folder
4. Click the extension icon, enter your Gemini or OpenAI API key

## How it works

Focus any text input, type something mean, click "Jerk Off". The extension rewrites it as something a decent person would say. Click accept to replace your draft.
