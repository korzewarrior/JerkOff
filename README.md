# JerkOff

**Stop being a jerk online.**

A Chrome extension that catches your angry comments and rewrites them into something a decent human would say. Powered by AI. Fueled by your rage.

## Demo

![Demo](docs/demo.gif)

## How It Works

1. **Type your angry comment** — Go ahead. Write that furious reply. Get it all out of your system.
2. **Click "Jerk Off"** — Hit the button that appears next to any text input. Your draft gets sent to AI for a full de-jerking.
3. **Post the nice version** — Review the rewrite. It keeps your intent but strips the venom. Click accept. Be the bigger person.

## Features

- **BYO API Key** — Use your own Gemini or OpenAI key. No middleman, no markup, no subscription fees.
- **Dual AI Support** — Choose between Google Gemini and OpenAI (GPT-4o-mini). Switch anytime from settings.
- **Works Everywhere** — Any text input on any website. Special optimization for Twitter/X replies.
- **Privacy First** — Zero data stored. Your API key stays in your browser. No analytics, no tracking, no servers.
- **Context-Aware** — Reads the post you're replying to so the rewrite actually makes sense in context.
- **One-Click Replace** — Accept the suggestion and it drops into the text field. Works with React, Vue, and contenteditable.

## Install

1. [Download the latest release](https://github.com/korzewarrior/JerkOff/releases/latest)
2. Unzip, go to `chrome://extensions`, enable Developer mode
3. Click "Load unpacked", select the folder
4. Click the extension icon, enter your Gemini or OpenAI API key
